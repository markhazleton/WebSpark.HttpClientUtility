# Guide

## Directory map

- .documentation/memory/ - active governance and memory artifacts
- .documentation/specs/ - active specifications and PR review records
- .documentation/commands/ - team prompt overrides
- .documentation/scripts/ - team script overrides
- .documentation/decisions/ - architecture decisions
- .documentation/releases/ - release documentation
- .documentation/copilot/ - generated audit/harvest reports

## Key files

- .documentation/memory/constitution.md - governing principles
- CHANGELOG.md - historical change and archive log

## How to use

- Stock prompts resolve from .devspark/defaults/commands/
- Team overrides resolve from .documentation/commands/
- Personal overrides resolve from .documentation/{git-user}/commands/
- Stock scripts resolve from .devspark/scripts/; team overrides in .documentation/scripts/

## Constitution location

- .documentation/memory/constitution.md

## What is in .archive/

- Completed and historical docs. Do not read from here during normal operations.
